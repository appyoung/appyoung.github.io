  • 建立一个电子商务网站、B2C商城需要多少钱？

    说道电子商务模式之一B2C，大家应该经常会在耳边听到。作为独立电商平台发展的大趋势之一，最近也一直备受青睐。就此以我们多年的开发经验聊一下，搭建一个B2C网站需要多少费用？

    首先要明确开发一个网站或者一个独立电商平台，成本是由哪几部分组成？

    1. 设计、开发成本

    这个其实也就是开发公司的成本，首先明确一点，下面说明的，仅仅是找第三方进行开发。这些成本的来源在于开发公司的规模和实力。大体上分为个人、小团队、小公司和大公司。不同开发方的规模不一样，设计开发成本也不一样。当然，这些都得符合自身的电子商务平台业务需求。开发方规模越大，分工越细，细节地方处理的越好，功能开发的越完善，也越有保障。大公司需要支付办公地点、规模庞大的开发人员、以及管理层的工资，广告推广费用，以及其他各种费用，所以对应下来，在需求相同的情况下每个项目的单价也越高。

    其次，看你开发的电子商务平台的要求和规模，例如：设计要求，功能要求，功能完善度，以及网站安全级别。设计要求高，需要安排单独的设计人员进行设计，要求低的，就可以直接进行模仿其他成熟网站。功能要求，涉及到你要开发多少模块，多少种营销方法。B2C平台少不了各种营销活动模式。每一种模式的设计、开发都是需要成本的。

    <1>个人电商模式

    个人独立管理整个网站，进行引流，用户进行注册购买商品。

    <2>平台模式

    电子商务网站提供平台，负责推广。让有良好货源的商家进行开店铺，发售商品，然后用户进行购买，这个就是传统的B2C模式。商家对商家像阿里巴巴的就是B2B模式。原理是一样的。像这种模式，在开发方来说，就需要开发网站商城（给用户看的），用户个人中心、网站管理后台、商家管理后台，4个大的版块。

    <3>分销、导购模式

    类似于阿里妈妈，通过淘宝客进行引流，方便用户抢占优惠，购买商品。本身不做商品平台，只做商品导购平台。

    <4>其他模式

    其他模式包括很多，网站本身的各种模式。包括抢红包优惠，代金券优惠，邀请优惠，答题有奖，秒杀活动，拼单活动，砍价活动，抽奖活动等等，个人商品博客分享。每一个模块都需要进行设计开发，里面涉及的关系错综复杂。大型平台型网站都是根据需求定预算，而不是直接定价格，直接定价格是不负责任的。

    2. 服务器成本

    服务器成本在初期是不需要多少费用的，基础配置，一年几千足以。做电子商务网站，不推荐用空间之类的，不要因为价格便宜而选择。想后期做大做强，独立的服务器是必不可少的，推荐使用阿里云。稳定而灵活。

    3. 接口、认证成本

    做电子商务平台型网站，还需要使用很多第三方服务，包括短信验证，部分数据接口，支付接口等等，以及部分平台的认证费用。有些是免费的，有些是付费的。拿支付宝接口来说，一般是需要有公司资质的对公账户。

    4. 域名成本

    域名分为直接注册域名和购买二手域名。直接购买域名成本低40-70元/年左右，好记的域名资源少。二手域名资源多，能找到性价比高的，但是对应的价格相应的也高，价格不定。

    5. 维护成本

    网站开发完成了，后期都是需要维护的，包括bug的修复，功能的小幅度修改。还有网站紧急问题处理等等。

    6. 更新迭代成本

    更新迭代，就包括网站功能的增加扩充，大幅度修改，以及节日活动版块的开发和经常性调整。不进行更新迭代，给用户以变动活力，不利于长期发展。

    综合以上的成本组成，再结合实际需求。

    小型电子商务网站：6000-2万左右

    中型电子商务网站：2万-10万左右

    大型电子商务平台网站：10万-无上限

    一般模板型电子商务网站：8000-5万左右

    以上是市场价格大体区间范围，具体的价格根据实际情况和市场因素而定，觉得有用的喜欢加分享呦。

  • 做一个网站大概需要多少钱？

      像我们做这方面已经有十多年了，其中遇到了许多形形色色的客户，其中直接问价的朋友也是非常多。在这里我就不和大家过多的拐弯抹角，就让我为你分析一下做一个符合自己需要的网站大概需要多少钱？

    1.需求

      第一步是需求，首先要明白自己要做哪种网站。如果是简单搭建一下个人网站，百度一下，就应该有免费的个人模板网站申请。如果是想自己练手学习建站方面的知识的话，购买一个域名和申请一个小的空间就可以满足自身的需求。以上这两种方法，一个是免费的，一个只需要域名和空间的费用。
      如果因为公司业务需求，暂不不考虑自己开发，那么就可以请外包公司，进行第三方开发，或者组建自己的开发团队开发。这两种方法都必须根据自身的发展需求进行选择。简单企业官网可以直接找第三方外包出去。简单应用类型的网站可以考虑前期先外包出去，后期上线运营了后，再转到自己开发团队。如果想完完全全开发自己的核心产品，并在资金充足的情况下，可以考虑组建自己的开发团队。

    2.价格

      一般一个网站由以下这几个部分组成，域名、服务器、程序代码。而其中的开发过程就包括项目的沟通、设计、代码开发、测试。一般新买的域名在50元/年不等。空间的费用在200-500元/年不等。大型网站考虑服务器800-几千元/年。找公司开发的话，应用开发的人工费用就占其中大头了。一般在1000元-50万不等。主要根据要求的高低，任务量的大小，和项目的复杂程度来定。

    3.结论

    个人娱乐型网站：免费
    个人研究型网站：300元/年左右
    一般企业官网：800-3000左右
    设计要求高官网：3000-几万不等
    cms二次开发网站：6000-几万不等
    论坛型网站：6000-几万不等
    模版型商城类型网站：8000-2万左右
    开发型商城、平台型网站：1万-十几万不等
    完全oem网站：几万-无上限
    微信公众号、小程序开发：几千到几万不等
    app开发：一般基本以万作单位

  • 刚做的官网，需要进行网站备案吗？

    在网站开发建设的过程中，始终绕不开的一个问题。就是网站是否真的需要备案？备案的好处在哪里？

    1.什么是备案？

    网站备案，其实就好比身份证号、车牌号、门牌号，给网站域名做一个身份标志。如果网站有一天出现了问题、违法了，知道要找谁，谁来负责这个事情。所以如果一个网站想在中国更好的运行下去，备案是必须的。备案的话，分为好多种。最为基本的是ICP备案。

    2.怎么备案？

    一般的云服务商、域名服务商都有进行备案的入口。如果不明白的话，可以直接要做网站的服务商帮你进行备案。备案可以和网站建设开发同时进行，在备案期间，备案域名是不允许访问的。备案是免费的。备案一般需要15-20天左右。不同的地区，备案通过的时间，也不一样。

    3.备案能带来哪些好处？

    不备案，在中国，就相当于黑户。一般备案了的网站，会让用户更加可信。备案的网站也有利于搜索引擎抓取，更好的进行展示营销，提高自己的品牌知名度。

  • 根据微信小程序功能判断所属服务类目，开发运营微信小程序需要哪些资质要求？

      首先附上微信小程序官方开放的服务类目网址：https://developers.weixin.qq.com/miniprogram/product/material.html，小程序开放的服务类目，会随着相关政策、法律法规以及平台要求的变化而产生变化。具体变化请看官方最新公布。
      随着微信小程序的日渐发展和完善，小程序的审核机制也变得越来越严格。抛开小程序开发的具体过程，这里主要说一下小程序开发完成后的类目归属以及审核方面的问题。但是，当你决定开发一款小程序的时候，你就需要知道你的小程序需要实现哪些功能以及功能与经营范围的匹配程度，因为具体的功能资质审核，小程序并没用完全死的规则。下面，主要以非个人主体小程序开放的服务类目为主要内容进行简单说明。

      1. 明确经营范围

      非个人主体小程序目前大多数主要的是企业，当然不排除组织和政府之类的。那么既然是企业，公司营业执照肯定是必不可少的。公司营业执照主要使用信息：公司名称、统一社会信用代码、经营范围、营业执照照片、对公账户、公司公章。还需要法人或者负责人的相关信息：姓名、身份证编号、身份证照片正反面、常用手机号、管理员微信等。
      其中经营范围主要确定了小程序实现的主要功能。例如小程序实现的功能是招聘兼职方面的。就需要有人力资源信息管理咨询服务的经营范围。小程序实现的是自营服装、鞋、箱包等功能，就需要具有服装、鞋、箱包售卖的经营范围。但是营业执照具有相应的经营范围仅仅是基础。微信官方对于特殊的行业要求，具有特殊的许可证要求。就像刚才所说的招聘兼职功能属于小程序商业服务中的招聘/求职（中介类）类目，就需要有《人力资源服务许可证》，而许可证申请应当依法向人力资源社会保障行政部门申请行政许可。

      2. 明确小程序功能需求

      弄好了经营范围还是不够的，你还需要根据你自身的条件来合理的开发你的小程序功能。例如小程序中实现的朋友圈功能，带有发圈，点赞，互动评论，支持文字、图片、视频发布。就相当于具有了社区、论坛功能。而社区、论坛功能属于互联网信息服务，国家政府对于互联网信息服务的管控还是比较严格的。如果小程序提供的信息交流服务不涉及盈利，就需要《非经营性互联网信息服务备案核准》。对于经营性交友、信息交流社交平台，就需要有《增值电信业务经营许可证》。如果你的小程序功能中涉及到文娱视频功能，还需要具有《信息网络传播视听节目许可证》或《广播电视节目制作经营许可证》或《网络文化经营许可证》或《广播电视节目播出机构许可证》

      3. 是否需要特殊资质，以及是否符合申请条件

      像上述所说，如果你的小程序做的是一个平台，涉及到商家加盟，商家拥有自己的店铺，可以自己发布自己的商品，然后统一在小程序上面进行展示交易，类似于京东、拼多多。那么你就属于电商平台，需要具有《增值电信业务经营许可证》资质。具体办理《增值电信业务经营许可证》需要哪些材料和条件，可以自行百度，或者向当地电信部门进行咨询。

      4. 替代方案

      当然，像很多小的创业公司刚刚起步，资金规模等各种条件都不够时，就可以使用替代方案进行解决。例如，你的小程序涉及商家平台、社区、论坛、视频功能，或者更复杂需求的代理模式、多级代理模式、分级返佣模式、用户裂变模式等，在没有资质的情况下，在小程序中都是不予以审核通过的，或者说一些特殊功能，不管有没有资质都是不予通过审核的。正是因为如此，常见的替代方案是：将所有的功能移植在公众号中，相当于是公众号应用，使用的技术也就是公众号微网页，基本上可以实现你想要的全部功能。

      5. 作弊不可取

      有很多时候，人们的想法是无穷的，有人甚至想到了，将关键功能，不能审核通过功能先隐藏起来，然后通过后台开关进行启用或关闭。其实这种作弊方法，微信官方早就已经想到了，在进行小程序审核的时候，微信将会将显示或者隐藏的功能进行全面的检查。
      当然你关键的功能还可以使用网页嵌套进行隐藏，在审核的时候，展示的是普通的内容。当审核通过后，再把网页中的核心不过审功能开启，这种方法在理论情况下是可以实现的。但是，一旦被人发现举报， 官方进行审理，那么换来的就是小程序永久封禁！

  • 企业需不需要做一款微信小程序？

    企业需不需要做一款微信小程序？首先这个问题得先得问一下自己。小程序是什么东西？了解吗？小程序目前的行情怎么样？小程序未来的发展趋势怎么样？企业需要、或者说适合做小程序吗？企业如果要做小程序，需要怎么做？做小程序，需要考虑盈利吗？这些都是做一款小程序之前需要想清楚的几个问题。下面我们就来重点分析一下什么情况下需要、什么情况下不需要？

    1. 行业定位

    首先要进行行业定位，企业是属于哪个行业的，目前这个行业有没有类似的小程序出现。有的话，目前该小程序的市场情况是怎样的。如果没有的话，新做一款本行业的小程序，符不符合市场需求，符不符合小程序的官方规定，符不符合国家政策。这每一个细节问题都决定着是否真的需要做一款小程序。

    2. 场景延伸

    企业做一款小程序，需要考虑到用户的使用场景。用户在怎样的场景下会用到该程序。使用的频率、使用的时长、使用的粘性是怎样的？用户在特定的环境下使用方不方便，使用的时候有没有特定的要求和顾忌。需不需要其他人进行配合使用。使用时的心情是怎样的？做之前，应该把这些问题想清楚。

    3. 用户传播

    企业做的小程序对于用户来说，易不易于使用和传播，传播的途径和方式有哪些？用户传播，需不需要成本，能给自己带来哪些好处？其实，一款小程序能解决特定群体的特殊问题，那么这款小程序就能很好的在该群体中形成共鸣，有一个恰当的点的话，就会形成爆发。

    4. 行为习惯

    还有一点比较重要的是，用户使用该小程序会不会形成一种行为习惯，是不是每天都必需要使用。或者当用户想要解决某个问题的时候，能恰好想到该小程序能解决此问题。如果可以，那证明这款小程序有被开发的需要。

    总的来说，企业想要做一款小程序前，需要把一些具体的问题想清楚。小程序的定位、小程序的使用场景、小程序的功能和界面风格、小程序的使用习惯、小程序的创意和推广等等。问题想没想好、方案策没策划好，都决定着小程序的必需性和方向性。

  • 怎么做一个商城小程序?

    如何开发一个商城小程序？怎么把自己的产品放在小程序上面卖？下面我们就来详细的学习一下。

    一、微信公众号

    1. 百度搜索微信公众号
    2. 立即注册，选择小程序
    3. 帐号信息，邮箱激活
    4. 信息登记，信息登记必须要选择企业主体，个人主体无法使用很多功能。登记需要准备好：营业执照、管理员身份证号码、对公银行账户等等
    5. 完善小程序基本信息

    二、小程序开发工具

    1. 学习文档：官方文档
    2. 打开微信web开发者工具，用管理员账号扫码登陆，选择小程序项目，点击右下角加号创建项目。
    3. 选择项目目录（小程序代码存放目录），填写appid（在登陆的小程序管理平台找到 管理 - 开发设置），填写项目名称。
    4. 确定后，就会跳到开发界面模式。
    5. 根据官方文档，编写小程序，完成JSON、WXML、WXSS、JS等的编写。
    6. 编译预览、可以实时测试开发效果。

    三、上线

    1. 提交审核，提交审核之前，可以先发布为体验版。所有功能测试无误、符合自己的要求后，再提交审核，一般审核都需要1到几个工作日不等。审核不通过，再次提交需要重写审核。
    2. 发布，审核通过后，就可以通过小程序的二维码进行进入使用了。还有其他的很多获取小程序途径，这里就不一一举例了。
    3. 开始使用

    四、商城

    上面三个是基本的小程序开发流程，然而商城的实现就是在步骤二中进行的。自己做一个商城，你首先需要勾画流程和逻辑，再进行界面设计，再到交互代码的编写，然后是和后端、服务器的数据联系。
    一般商城会包括以下几个模块：商品、店铺、购物车、个人中心、订单、物流、消息、营销、聊天、支付等。所有的功能实现了，一个商城也就出来了。

    想要开发一个完整的、用户体验良好的、能为自己带来订单的商城小程序，是非常不容易的。如果，觉得自己开发麻烦，在资金充足的情况下。可以将专业的事情交给专业的人来进行解决。你只需提供需求、想法，让开发者们开发即可。
